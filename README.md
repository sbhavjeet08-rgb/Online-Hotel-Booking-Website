# Online Hotel Booking Website

A full-stack online hotel booking web application.

## Features
- User registration & login with validation
- JWT authentication
- Hotel listing and booking
- View bookings
- Admin hotel management

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MySQL
- Auth: JWT

## Project Structure
backend-server  
frontend-app/hotel-app  

## How to Run

### Backend
npm install  
npm start  

### Frontend
npm install  
npm start
